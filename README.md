# PageRank and HITS Algorithms

Information Retrieval project for ranking web pages using markov chains etc.

## To install dependencies :

`$ pip install numpy`

`$ pip install scipy`

`$ pip install networkx`

## To run PageRank

`$ python pageRanker.py`

## To run HITS

`$ python HITS.py`
